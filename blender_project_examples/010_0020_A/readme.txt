README

This folder contains files that make up a shot of Sprite Fright.

    - Open the .anim file if you want to check out the lighter animation setup
    - Files with the suffix .FX contain simulation or setups to generate various effects within the shot.
    - the .lighting file links in the animation + effects, and contains the full render setup to output the final image
